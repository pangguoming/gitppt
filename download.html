# NeuroDB Release 1.0.0 (2023-09-10)

  1. Significant performance improvement in pattern matching algorithm.
  2. Reconstruction and optimization of shortest path retrieval algorithm.
  3. Enhanced functionality for importing CSV data.
  4. Reconstruction of sorting algorithm.
  5. Time-out fallback mechanism for full path retrieval.
  6. Modification of default automatic persistence storage strategy.
  7. Optimization of log level configuration and output.
  8. Addition of a set of unit test case scripts.
  9. Bug fixes and memory leak resolutions in the code.
